# Remote-sonoff-POWR2
## Notes:
- This project was built using Platform IO
- By opening the project file (platformio.ini) with Platform IO, all dependencies will automatically be installed and device will be pre-configured to upload.

- ESP8266WiFi
- ESP8266WebServer
